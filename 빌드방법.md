# 한글 뷰어 안드로이드 앱 — 빌드 방법

`.hwp / .hwpx` 파일을 탭하면 광고 없이 바로 열리는 안드로이드 앱입니다.
파싱 라이브러리를 **앱에 내장**해 인터넷 없이 동작하며, 문서는 폰 안에서만 처리됩니다(서버 전송 없음).

이 폴더는 **빌드 준비가 끝난 소스**입니다. 아래 두 방법 중 하나로 설치용 APK 파일을 만들 수 있습니다.
(작업 환경에 안드로이드 SDK가 없어 완성된 APK 파일 자체는 동봉하지 못했습니다 — 마지막 빌드 한 단계만 아래 방법으로 진행하면 됩니다.)

---

## 방법 A. GitHub Actions — 설치 없이 자동 빌드 (권장)

컴퓨터에 아무것도 설치하지 않고, 이 소스를 GitHub에 올리면 자동으로 APK가 만들어집니다.

1. github.com 가입(무료) 후 새 저장소(repository) 생성 — 이름 예: `hwp-viewer`
2. 이 `한글뷰어_앱` 폴더 안의 **모든 파일**을 저장소에 업로드
   (웹에서 "Add file → Upload files"로 폴더째 끌어다 놓으면 됩니다)
3. 업로드하면 자동으로 빌드가 시작됩니다. 저장소 상단 **Actions** 탭 → 최신 실행 클릭
4. 초록색 체크(완료)가 뜨면 그 화면 아래 **Artifacts → `hwp-viewer-apk`** 를 내려받기
5. 압축을 풀면 `app-debug.apk` — 이 파일을 폰으로 옮겨 설치

> 빌드는 보통 3~5분 걸립니다. `.github/workflows/build.yml` 에 빌드 설정이 들어 있습니다.

---

## 방법 B. 안드로이드 스튜디오 — 내 컴퓨터에서 빌드

1. [Android Studio](https://developer.android.com/studio) 설치(무료)
2. `File → Open` 으로 이 `한글뷰어_앱` 폴더 열기 (Gradle 동기화가 자동 진행됩니다)
3. 상단 메뉴 `Build → Build App Bundle(s) / APK(s) → Build APK(s)`
4. 완료 후 뜨는 알림의 `locate` 클릭 → `app/build/outputs/apk/debug/app-debug.apk`
5. 이 APK를 폰으로 옮겨 설치

---

## 폰에 설치하기 (사이드로드)

- APK 파일을 폰으로 옮긴 뒤 파일 관리자에서 탭 → 설치
- "출처를 알 수 없는 앱" 경고가 뜨면 해당 권한을 허용해야 설치됩니다
  (설정 → 앱 → 특정 파일 관리자/브라우저에 "이 출처 허용")
- 설치 후, 파일 관리자나 메일에서 `.hwp` 파일을 탭하면 **"한글 뷰어"로 열기** 선택지가 나옵니다

## 구성 (참고)

```
한글뷰어_앱/
├─ app/src/main/
│   ├─ AndroidManifest.xml      ← .hwp/.hwpx 열기 연동(인텐트 필터)
│   ├─ java/com/hwpviewer/MainActivity.java   ← WebView + 파일 수신
│   └─ assets/
│       ├─ viewer.html          ← 뷰어 본체(오프라인)
│       └─ lib/                 ← 파싱 라이브러리 내장(cfb·fflate·aes-js)
├─ build.gradle, settings.gradle, gradle.properties
└─ .github/workflows/build.yml  ← 자동 빌드 설정
```

- **디버그 APK**는 서명 없이 개인 설치가 가능합니다(개인용으로 충분).
  Play 스토어 배포용 정식 서명본이 필요하면 별도 서명 키가 필요합니다.
- 뷰어 로직을 고치려면 `assets/viewer.html` 만 교체하면 됩니다.

## 지원 범위

- 일반 HWP, **배포용 HWP(암호화 문서)**, HWPX 모두 지원 — 텍스트·표(격자·병합) 복원
- 그림 배치·수식·글꼴 색 등 세부 서식은 단순화될 수 있습니다(텍스트·표 중심 뷰어)
